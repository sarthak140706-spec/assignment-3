Task 1: Factorial Function

Objective:
To create a Python program that calculates the factorial of a given number using a function.

Problem Statement:

Define a function named factorial that accepts a number as an argument.

Calculate the factorial of the number using a loop or recursion.

Return the calculated factorial.

Call the function with a sample number and display the result.

Expected Output Example:
If the input number is 5, the program should return:

Factorial of 5 is 120


Learning Outcomes:

Understanding the concept of functions in Python.

Learning how to use loops and recursion.

Applying logic to solve mathematical problems programmatically.
